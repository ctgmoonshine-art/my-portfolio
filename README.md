# Сайт-визитка | Портфолио

Веб-сайт-визитка, созданная в рамках производственной практики по изучению Visual Studio Code.

## Описание проекта

Проект представляет собой адаптивный веб-сайт-портфолио, включающий:
- Главную страницу с приветствием
- Раздел "Обо мне"
- Раздел "Навыки" с иконками технологий
- Раздел "Проекты"
- Контактную информацию

## Используемые технологии

- HTML5 (семантическая разметка)
- CSS3 (Flexbox, Grid, CSS-переменные, медиазапросы)
- Font Awesome (иконки)
- Visual Studio Code (редактор)
- Git (контроль версий)

## Особенности реализации

1. **Адаптивный дизайн**: Корректное отображение на всех устройствах
2. **Современный CSS**: Использование CSS-переменных, Flexbox, Grid
3. **Доступность**: Семантическая HTML-разметка
4. **Производительность**: Оптимизированные изображения и код

## Как запустить проект

1. Скачайте или клонируйте репозиторий
2. Откройте файл `index.html` в любом современном браузере
3. Или используйте Live Server в VS Code для автоматической перезагрузки
